Automatically set up all customizations. Simply git clone and run ./install.sh

Sets up:
- Dotfiles
- Vim
- tmux
- dwm
- st
- slstatus
- fonts
- zsh (oh-my-zsh,custom theme), bash prompts
- compiler toolchains
- xorg configs
- git
- Virtualbox support
- Arch support
- Latex
- Browser

Warns on uninstalled dependencies.
