Automatically set up all customizations. Simply git clone and run ./install.sh

Sets up:
- Dotfiles
- Vim
- tmux
- dwm
- st
- slstatus
- fonts
- fish (oh-my-fish), zsh (oh-my-zsh,custom theme), bash prompts
- zsh-autosuggestions
- compiler toolchains
- xorg configs
- git
- Virtualbox support
- Arch support
- Latex
- Browser
